# PalacesMapApp
India Palaces Map App (SwiftUI + MVVM)


This iOS application displays popular places in India on a map using static data. Built using SwiftUI and MVVM architecture to demonstrate clean code structure and map integration.

**Features:**
  *  Display locations on Map
  * Custom annotations
  * List of places
  * Static JSON/local data handling

**Tech Stack:**
  * Swift
  * SwiftUI
  * MVVM
  * MapKit
